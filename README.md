# agentic-lite

## Installation

# ⚠️ Renamed — use agentic-core

agentic-lite has been renamed to [agentic-core](https://github.com/momomo-agent/agentic-core).

```bash
npm i agentic-core
```

```js
const { agenticAsk } = require('agentic-core')
```

See [agentic-core docs](https://momomo-agent.github.io/agentic-core/) for details.
